The <p> Element

The <p> element is used to define paragraphs. Web browsers will typically insert white space between paragraphs, but the spacing (along with other formatting) can be modified with CSS.

Note: If you want to line breaks within paragraphs, or between lines of text that don't belong in paragraphs, you should use the <br> element instead.
