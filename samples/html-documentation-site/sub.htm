The <sub> Element

The <sub> element allows you to display subscript text. As with other text formatting elements like <i> and <strong>, you should only use this tag for semantic purposes, such as mathmatical/chemical notations or footnotes.

Example:

                    
                        <p class="example-code">
                            The chemical formula for water is: H<sub>2</sub>O.
                        </p>
                        <p class="example-code">
                            The Fibonacci number formula is: F<sub>0</sub> = 0 and F<sub>1</sub> = 1, and then:
                            F<sub>n</sub> = F<sub>n-1</sub> + F<sub>n-2</sub>.
                        </p>

Result:

The chemical formula for water is: H2O.

The Fibonacci number formula is: F0 = 0 and F1 = 1, and then: Fn = Fn-1 + Fn-2.

For more details, please see the wiki page for this element at w3c.org.
