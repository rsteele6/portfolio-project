The font-family Property

The font-family property is used for specifying the font for an element. Because all browsers won't necessarily support all fonts, you can specify several fonts in this property as "backups." If the user's browser can't render your first choice, it'll try the second, third, and so on. Therefore, it's best to start with the exact font you want, and end with a generic font family that the user's browser can choose from if none of the others are available.

Example:

                    
                        #example-code-font-family-cs
                        {
                            font-family: "Comic Sans MS", cursive, sans-serif;
                        }

                        #example-code-font-family-lc
                        {
                            font-family: "Lucida Console", Monaco, monospace;
                        }
