The color Property

The color property sets the color of text within an element. You can specify color values using HEX, RGB, RGBA, HSL or HSLA values, or by specifying a color name (over 100 are supported).

Example:

                    
                        #example-code-font-color-named
                        {
                            color:red;
                        }

                        #example-code-font-color-hex
                        {
                            color: #004d00;
                        }

                        #example-code-font-color-rgb
                        {
                            color: rgb(77, 148, 255)
                        }
