The <ol> Element

The <ol> Element, along with nested <li> Elements allow you to create ordered lists. The list will be numbered by default, but you can use the <type> attribute to letters or Roman numerals.

Example:

                    
                        <small class="example-code">Fig 1</small>
                        <ol>
                            <li>Feed the cat</li>
                            <li>Pay bills</li>
                            <li>Go grocery shopping</li>
                        </ol>

                        <small class="example-code">Fig 2</small>
                        <ol id="ol-alpha-upper">
                            <li>Feed the cat</li>
                            <li>Pay bills</li>
                            <li>Go grocery shopping</li>
                        </ol>

                        <small class="example-code">Fig 3</small>
                        <ol id="ol-alpha-lower">
                            <li>Feed the cat</li>
                            <li>Pay bills</li>
                            <li>Go grocery shopping</li>
                        </ol>

                        <small class="example-code">Fig 4</small>
                        <ol id="ol-roman-upper">
                            <li>Feed the cat</li>
                            <li>Pay bills</li>
                            <li>Go grocery shopping</li>
                        </ol>

                        <small class="example-code">Fig 4</small>
                        <ol id="ol-roman-lower">
                            <li>Feed the cat</li>
                            <li>Pay bills</li>
                            <li>Go grocery shopping</li>
                        </ol>
