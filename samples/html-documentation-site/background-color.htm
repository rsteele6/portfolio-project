The background-color Property

The background-color property sets the background color of an element. As with the color property, you can specify color values using HEX, RGB, RGBA, HSL or HSLA values, or by specifying a color name.

Example:

                    
                        #example-code-background-color-named 
                        {
                            background-color: red;
                        }

                        #example-code-background-color-hex
                        {
                            background-color: #004d00;
                        }

                        #example-code-background-color-rgb
                        {
                            background-color: rgb(77, 148, 255)
                        }
