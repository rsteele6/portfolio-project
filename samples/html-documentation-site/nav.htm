The <nav> Element

The <nav> element is used to define a major block of navigation links on a webpage, such as a navigation bar. It should NOT be used for all links on a page.

You can create a basic menu of links, as in figure 1, or you can nest other elements (such as ul) within it to create more ornate menus, as in figure 2.

Example:

Fig 1. 
                    
                        <nav class="example-code-nav">
                        <a href="#">Home</a> |
                        <a href="#">About Us</a> |
                        <a href="#">Careers</a> |
                        </nav>


Fig 2. 
                    
                        <nav id="example-code-navbar">
                            <ul>
                                <li><a href="#"><span class="fas fa-home fa-lg"></span></a></li>
                                <li>
                                    <a href="#">About</a>
                                    <ul class="navbar-hide-submenus">
                                        <li><a href="#">Company History</a></li>
                                        <li><a href="#">Mission Statement</a></li>
                                        <li><a href="#">Contact Form</a></li>
                                    </ul>
                                </li>

                                <li>
                                    <a href="#">Careers</a>
                                    <ul class="navbar-hide-submenus">
                                        <li><a href="#">Benefits</a></li>
                                        <li><a href="#">Available Positions</a></li>
                                    </ul>
                                </li>
                            </ul>
                        </nav>

Result:

Fig 1.
Home | About | Careers
Fig 2.
  • About
    • Company History
    • Mission Statement
    • Contact Form
  • Careers
    • Benefits
    • Available Positions
