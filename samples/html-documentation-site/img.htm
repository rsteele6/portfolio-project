The <img> Element

The <img> element is a container used for rendering a linked image. It has many available attributes, but two are required: src and alt. Src is used for specifying the location of the linked resource, and alt is used for providing a textual description of the image if the user's browser cannot render it, or if the resource is unavailable.

Example:

                    
                        <h3 class="example-code">Here are Two Images</h3>

                        <img class="example-code" src="images/cat-developer-meme.jpg" alt="Developer Cat Meme">
                        <img class="example-code" src="images/cat-papertowels-meme.jpg" alt="Paper Towel Cat Meme">

Result:

Here are Two Images

For more details, please see the wiki page for this element via w3c.org.
