Introduction to HTML5 and CSS3

HTML5 is the standard markup language used for building web pages, and CSS is a styling language used for adding visual formatting to HTML documents as well as programs written in other languages such as Java. This site goes over a few basic HTML elements and CSS properties to help you get started creating your own websites.
