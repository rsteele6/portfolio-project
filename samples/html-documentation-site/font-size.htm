The font-size Property

The font-size property allows you to set the size of your text. The sizing can be in absolute units (such as px) or relative units such as em (resizes font relative to the current font size) or percent, which resizes the text to a percentage of the parent element font size.

Note: font-size should be used in conjunction with the appropriate heading elements where appropriate. It should not be used on its own to denote headings.

Example:

                    
                        #example-code-font-size-h2
                        {
                            font-size:20px;
                        }

                        #example-code-paragraph
                        {
                            font-size:0.75em;
                        }
