The <h> Element

The <h> element is used for identifying document or section headings. There are six levels of headings, with <h1> being the most important, and <h6> the least.

While heading tags do confer some text styling on their own, they're strictly for identifying content headings and should not be used to style text.

Example:

                    
                        <h1 class="example-code">Heading 1</h1>
                        <h2 class="example-code">Heading 2</h2>
                        <h3 class="example-code">Heading 3</h3>
                        <h4 class="example-code">Heading 4</h4>
                        <h5 class="example-code">Heading 5</h5>
                        <h6 class="example-code">Heading 6</h6>

Result:

Heading 1

Heading 2

Heading 3

Heading 4

Heading 5
Heading 6

For more details, please see the wiki page for this element at w3c.org.
