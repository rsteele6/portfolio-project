The <table> Element

The <table> element allows you to arrange data into rows and columns of cells, similiar to a spreadsheet. You can define table headings with the <th> tag, which are bold and centered by default. The <tr> tag is used for creating table rows, and row data is enclosed in nested <td> tags. The example below uses CSS styling to create black borders and a white background.

Example:

                    
                        <table>
                            <tr>
                                <th>Pet Name</th>
                                <th>Species</th>
                                <th>Age</th>
                            </tr>

                            <tr>
                                <td>Sammy</td>
                                <td>Feline</td>
                                <td>11</td>
                            </tr>

                            <tr>
                                <td>Sasha</td>
                                <td>Feline</td>
                                <td>9</td>
                            </tr>

                            <tr>
                                <td>Odin</td>
                                <td>Canine</td>
                                <td>5</td>
                            </tr>
                            <tr>
                                <td>Miranda</td>
                                <td>Canine</td>
                                <td>10</td>
                            </tr>
                        </table>

Result:

Pet Name Species Age
Sammy Feline 11
Sasha Feline 9
Odin Canine 5
Miranda Canine 10

For more details, please see the wiki page for this element at w3c.org.
